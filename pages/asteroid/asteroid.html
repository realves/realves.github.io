Desvie dos asteróides!


Jogo básico para aprender a fazer a colisão de 2 objetos em Javascript.


Justificativas

Este jogo fez parte de um exercício proposto pelo meu professor de Programação Web, Pedro Cacique. O objetivo deste projeto era o de criar o método de colisão entre dois objetos circulares.


Programas utilizados

  • Brackets
  • Piskel

Referências

Nenhum jogo foi de fato utilizado como referência para este projeto, portanto darei os créditos para os criadores das imagens utilizadas aqui.

  • Asteróides: Open Game Art
  • Fundo estrelado: Artista desconhecido
  • Nave: PNGkey
  • Explosão: TOPpng
  • Corações: Criação própria

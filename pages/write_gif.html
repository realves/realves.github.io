Escrevendo o nickname


Pequena animação, no formato GIF, em que o nickname StyleDead é escrito.


Justificativas

O meu objetivo com este projeto foi o de aprender a criar um arquivo no formato GIF através do GIMP, já que eu nunca tinha feito um antes. Além de obter um conteúdo inicial para o meu portfólio.


Programas utilizados

Referências
